== Description ==

Checks whether Contact Form 7 is used on a page or post, and add contact form 7 js/css files only if contact form is used on that page. 

== Installation ==

= Manual Installation =
1. Unzip and upload the `rufo-remove-unused-files-of-cf7` folder to the `/wp-content/plugins/` directory.
2. Activate the Rufo Remove Unused Files of Contact Form 7 plugin through the ‘Plugins’ menu in WordPress.
